# Contributing to CS Vertex

We welcome contributions! Please follow these guidelines:

1. **Branching Strategy**: Create a feature branch from `main` (e.g., `feature/awesome-new-ui`).
2. **Commit Messages**: Write clear, descriptive commit messages.
3. **Pull Requests**: Submit PRs against the `main` branch. Ensure `npm run build` and `npm run lint` pass locally before pushing.
4. **Code Quality**: Ensure zero TypeScript errors and zero ESLint warnings.

Thank you for contributing to CS Vertex!
